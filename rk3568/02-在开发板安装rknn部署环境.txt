1.在开发板中安装miniconda环境
0. 配置WiFi：fltest_wifi.sh -i mlan0 -s wifi名称 密码(网线连接的可以忽略）

1.1 将安装包拷贝到开发板中，运行安装包，等待安装完成（最后一步建议选择no）: ./Miniconda3-latest-Linux-aarch64.sh

1.2 更新环境变量：source ~/miniconda3/bin/activate(每次开机之后都需要更新才能激活conda环境）

1.3 创建rknn环境：conda create -n rknn python=3.7
a. 这里可能会报错，说下载的通道中找不到满足条件的安装包，此时需要将.condarc中的下载源删除，更换其他源，这里更换了北京外国语大学开源软件镜像站：
channels:
  - defaults
show_channel_urls: true
channel_alias: https://mirrors.bfsu.edu.cn/anaconda
default_channels:
  - https://mirrors.bfsu.edu.cn/anaconda/pkgs/main
  - https://mirrors.bfsu.edu.cn/anaconda/pkgs/free
  - https://mirrors.bfsu.edu.cn/anaconda/pkgs/r
  - https://mirrors.bfsu.edu.cn/anaconda/pkgs/pro
  - https://mirrors.bfsu.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.bfsu.edu.cn/anaconda/cloud
  msys2: https://mirrors.bfsu.edu.cn/anaconda/cloud
  bioconda: https://mirrors.bfsu.edu.cn/anaconda/cloud
  menpo: https://mirrors.bfsu.edu.cn/anaconda/cloud
  pytorch: https://mirrors.bfsu.edu.cn/anaconda/cloud
  simpleitk: https://mirrors.bfsu.edu.cn/anaconda/cloud
b. 随后再次创建rknn环境：conda create -n rknn python=3.7
c. 创建完成后，激活rknn环境：source activate rknn

1.4 安装rknn_toolkit_lite2-1.4.0-cp39-cp39-linux_aarch64：
a. 将rknn_toolkit_lite2-1.5.2-cp37-cp37m-linux_aarch64.whl安装包拷贝到开发板中
b. 执行安装命令：pip install rknn_toolkit_lite2-1.5.2-cp37-cp37m-linux_aarch64.whl  -i https://pypi.mirrors.ustc.edu.cn/simple/
可能会报错： ERROR: Failed building wheel for psutil
c. 根据报错内容，下载安装psutil：conda install psutil
d. 随后重新安装：pip install rknn_toolkit_lite2-1.4.0-cp39-cp39-linux_aarch64.whl  -i https://pypi.mirrors.ustc.edu.cn/simple/

1.5 安装模型推理需要的库：
a. sklearn: conda install scikit-learn
b. pandas: conda install pandas
c. openpyxl: conda install openpyxl

至此，rknn部署的开发板环境已经安装完成
